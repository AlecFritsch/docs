## What is Havoc?

Havoc is a SaaS platform for AI agencies that deploy and manage **autonomous AI agents** for their clients. Built on [OpenClaw](https://docs.openclaw.ai), each agent runs in its own secure Docker container with gateway access to WhatsApp, Telegram, Discord, Slack, and more.

## Core Concepts

| Concept | Description |
|---|---|
| **Agent** | An autonomous AI entity running in a secure Docker container with its own OpenClaw gateway. |
| **Channel** | A messaging platform connection (WhatsApp, Telegram, Discord, Slack, Signal, etc.). |
| **Provider** | An AI model provider (Anthropic, OpenAI, Google, etc.) with API key management. |
| **Workspace** | Agent configuration files (AGENTS.md, SOUL.md, TOOLS.md, IDENTITY.md, etc.). |
| **Session** | A conversation thread between a user and an agent. |
| **Template** | Pre-built agent configurations for common use cases. |

## Architecture

```
Your Clients
    |
    v
[ WhatsApp / Telegram / Discord / Slack / ... ]
    |
    v
[ Havoc Platform ]
    |
    v
[ Agent Container 1 ] [ Agent Container 2 ] [ Agent Container N ]
    |                       |                       |
    v                       v                       v
[ OpenClaw Gateway ]  [ OpenClaw Gateway ]  [ OpenClaw Gateway ]
```

Each agent is fully isolated in its own Docker container with:
- Dedicated OpenClaw gateway on a unique port
- Per-agent workspace, memory, and session state
- Independent channel connections and provider configuration
