## Documentation Launch

We are excited to launch the official Havoc documentation. This site covers everything you need to get started with the platform:

- **Getting Started** — Quickstart guide and platform overview
- **Concepts** — Agents, Channels, Providers, and Workspaces explained
- **Guides** — Step-by-step tutorials for common tasks
- **API Reference** — Full OpenAPI documentation with 80+ endpoints

## Platform Highlights

- Deploy autonomous AI agents in secure Docker containers
- Connect to WhatsApp, Telegram, Discord, Slack, Signal, and more
- Support for 20+ AI model providers including Anthropic, OpenAI, and Google
- Agent workspace system with persona, memory, and tool configuration
- Real-time analytics and session management
- Team collaboration with role-based access control
- RESTful API with Clerk JWT and API key authentication
